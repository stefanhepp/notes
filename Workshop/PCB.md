PCB Design Notes
================

Board Design Rules
------------------

### Prototyping Boards
- Hole diameter: 1.02mm
- Pad size: 2.2mm

### CNC milled PCBs
- Tools
    - V-carve bit: 0.1mm
    - Drill: 0.8mm / 1.0mm
    - End mill: 1mm-2mm
- All through-holes: 0.8mm / 1.0mm
- Vias
    - Hole Diameter: 0.8mm (0.6mm wires)
    - Pad size: 2mm
- Traces
    - Trace minimum size: 0.3mm
    - Trace-to-trace clearance: 0.2mm
    - Pour-to-trace clearance: 0.3mm
